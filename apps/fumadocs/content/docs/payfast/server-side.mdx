The `Payfast` class provides typed wrapper methods that delegate to the component's Convex functions. Use it in your actions, mutations, and HTTP handlers.

## Instantiation

```ts
import { Payfast } from "@bazileros/payfast";
import { components } from "./_generated/api";

const pf = new Payfast(components.payfast, {
  sandbox: true,                   // Use sandbox.payfast.co.za
  getUserInfo: async (ctx) => {
    const identity = await getAuthIdentity(ctx);
    return { userId: identity.subject };
  },
});
```

### Options

| Option | Default | Description |
|---|---|---|
| `sandbox` | `false` | Use PayFast sandbox environment |
| `getUserInfo` | `undefined` | Callback to auto-resolve `userId` from Convex context |

## `getUserInfo`

When set, methods that accept `userId` resolve it automatically from the action context. Callers can omit the `userId` parameter.

## Methods

### `createCheckoutSession(ctx, args)`

Generates a signed checkout form. Returns `{ formActionUrl, formFields }`.

### `listTransactions(ctx, filters?)`

Queries the `transactions` table. Filters: `userId`, `status`, `limit`.

### `getTransaction(ctx, transactionId)`

Gets a single transaction by ID.

### `listSubscriptions(ctx, filters?)`

Queries the `subscriptions` table. Filters: `userId`, `status`.

### `getSubscription(ctx, token)`

Gets a single subscription by token.

### `pauseSubscription(ctx, args)`

Pauses a subscription via PayFast REST API.

### `unpauseSubscription(ctx, args)`

Resumes a paused subscription.

### `cancelSubscription(ctx, args)`

Cancels a subscription immediately.

### `updateSubscription(ctx, args)`

Updates subscription amount, frequency, cycles, or run date.

### `getCardUpdateUrl(token, returnUrl?)`

Generates a URL for the user to update their card details. Does not require Convex context — pure string construction.
